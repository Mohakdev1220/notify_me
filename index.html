Supabase — auto sync Live
Setup: supabase.com → New project → SQL Editor → run the SQL below → Settings → API → copy URL + anon key.

Update timetable from anywhere: Supabase dashboard → Table Editor → officesync_timetable → edit row → app updates automatically within 3 seconds.
SQL to run once in Supabase SQL Editor
create table if not exists officesync_timetable (
  id uuid default gen_random_uuid() primary key,
  device text not null unique,
  data text not null,
  updated_at timestamptz default now()
);
alter table officesync_timetable enable row level security;
create policy "Allow all" on officesync_timetable
  for all using (true) with check (true);
